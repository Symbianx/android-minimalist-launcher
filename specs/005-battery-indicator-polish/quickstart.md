# Quickstart: Battery Indicator Polish (Spec 005)

## Prerequisites
- Android Studio (latest)
- Kotlin, Jetpack Compose

## Steps
1. Checkout branch `005-battery-indicator-polish`
2. Open project in Android Studio
3. Implement/modify Compose UI for battery indicator in `app/src/main/java/...`
4. Add animation for charging state
5. Write/verify tests in `app/src/androidTest/` and `app/src/test/`
6. Run on device/emulator, verify indicator and animation

## Test
- Use Compose UI Test to verify indicator arc and animation
- Use ADB to simulate battery/charging state changes
